## About Bckgrnd

[Bckgrnd](https://bckgrnd.one/) is an all-in-one tool created for design freelancers and agencies to efficiently manage their clients, projects, tasks and files. It allows you to share your work with clients, receive feedback and track progress.

## ✨ Features

- 👥 Client management
- 📁 Project & task management
- 🗄️ File storage & sharing
- 📝 Review & approval workflow
- 🤝🏼 Live collaboration
- 📊 Reporting and analytics
